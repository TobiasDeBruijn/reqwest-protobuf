# Reqwest-protobuf
Bringing protocolbuffer support to reqwest.

# Sending protobuf
use the `protobuf` method on the `RequestBuilder` type

# Receiving protobuf
1. Accept it: `accept_protobuf` (on `RequestBuilder`)
2. Decode it: `protobuf` (on `Response`)
